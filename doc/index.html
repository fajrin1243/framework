Documentation Section

Structure Folder

[App] Setiap file website atau aplikasi anda terletak disini
[config] Configurasi dapat anda lakukan disini terdapat 3 file, config, route
[include] Lettakan setiap file js, css, images, dan lainnya pada include
[system] File utama sinc framework [log] digunakan untuk file log [tmp] digunakan sebagai temporary

Getting Start

Sinc Framework adalah framework php yang menggunakan metode mvc, untuk memulai lakukan configurasi pada folder config, kemudian buatlah controller, model, ataupun view pada folder app. anda bebas membuat structure file anda sendiri. hanya saja gunakan extends di setiap class yang anda buat.
Controller
class contoh_controller extends controller
Model
class contoh_controller extends controller
untuk kedua diatas tambahkan
function __construct()
{
parent::__construct();
}
View
untuk menargetkan atau mengarahkan controller dengan view anda gunakan parent::view('file_view_anda');

Library

Our library seperate to 3 :
- getcommon (hal yang biasa anda gunakan seperti session, dan function php pada umumnya)
- getutility (library yang anda bisa gunakan yang telah tersedia pada framework)
- gethtml (segala sesuatu yang dapat anda gunakan untuk menuliskan html seperti table, include css, js, image, url dan lainnya)

Extension

Umumnya setiap extension yang terindex pada framework terletak pada [system/ext] dan dapat anda akses menggunakan
getext::ext('nama_extension');
Untuk menambahkan extension secara manual dapat anda letakkan pada folder include
